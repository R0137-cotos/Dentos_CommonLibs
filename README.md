# バージョン採番方法

build.gradleファイルのリビジョン番号(pom.version)を＋1して「gradle uploadArchives」コマンドを実行し、
repositoryディレクトリに生成されたファイルも合わせてコミットすること。
※CommonLibs直下にあるbuildディレクトリを削除してから上記コマンドを実行すること。
